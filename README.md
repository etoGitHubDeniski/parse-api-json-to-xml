# Parse neometria

Команда для парсинга - ```php artisan parse:neometria```

Файлы:
- /app/Services/ParseDataClient.php - Клиент
- /app/Console/Commands/ParseNeometriaCommand.php - Консольная команда
- /output.xml - Сгенерированный xml файл
